# Lilia Motors – Luxury Car Showroom Website

A modern and fully responsive frontend website for a luxury car showroom, designed to deliver a premium user experience and visually reflect the high-end nature of the automotive industry.

---

## 📌 Overview

This project represents a complete frontend implementation of a car showroom website.  
The focus was on clean layout, smooth user interaction, and a strong visual identity that matches the concept of luxury vehicles.

The site is structured into multiple sections that work together to guide the user through the content in a clear and intuitive way.

---

## 🧩 Sections Included

- **Hero Section**
  - Strong visual introduction with headline, call-to-action buttons, and featured car image
  - Balanced layout between content and visuals

- **Features Section**
  - Highlighting key strengths using interactive cards
  - Hover effects and subtle animations

- **Services Section**
  - Step-based layout with active states and smooth transitions
  - Structured presentation of services

- **About Section**
  - Combination of image and text with clean alignment
  - Focus on readability and spacing

- **Portfolio (Cars Showcase)**
  - Grid-based layout for displaying vehicles
  - Filter system to switch between categories dynamically

- **Team Section**
  - Team members displayed with hover interactions
  - Social icons appear on interaction

- **Pricing Section**
  - Clear pricing cards with emphasis on readability
  - Visual hierarchy for plans and features

- **Blog Section**
  - Article cards with images, dates, and summaries
  - Structured layout for content display

- **Contact Section**
  - Simple and clean form design
  - Frontend validation for inputs

---

## 🎨 Design Approach

- Dark theme for a premium and modern look  
- Consistent spacing and alignment across sections  
- Use of shadows and gradients to enhance depth  
- Smooth transitions and hover effects without affecting performance  

---

## ⚙️ Technical Details

- Built using:
  - HTML5
  - CSS3
  - JavaScript (Vanilla)

- Layout techniques:
  - Flexbox
  - CSS Grid

- Features:
  - Fully responsive across all screen sizes
  - Scroll-based animations (reveal on scroll)
  - Interactive UI elements (hover, active states)
  - Clean and structured code

---

## 📱 Responsiveness

The website is fully responsive and optimized for:

- Mobile devices  
- Tablets  
- Desktop screens  

All elements were adjusted carefully to maintain layout consistency and usability across different screen sizes.

---

## 🎯 Key Focus Areas

- User experience and clarity of navigation  
- Visual balance between content and images  
- Smooth interaction and subtle animations  
- Performance and clean structure
